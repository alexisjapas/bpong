1. **Tests (0%)** : Aucun test. C'est le levier d'amelioration le plus impactant (poids 20%)

4. **CI incomplete** : Aucun check sur les PR (lint, build, tests) — uniquement du CD sur tags

2. Etoffer le README (description, screenshot, instructions d'installation/utilisation)
3. Ajouter un workflow CI sur les PR (cargo clippy, cargo fmt --check, cargo test)
4. Corriger le bug de paddle-tunneling (item roadmap)
5. Adopter les conventional commits de maniere stricte
6. Creer des issues GitHub pour tracker les items de la roadmap
